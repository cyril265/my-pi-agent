# Vendored sq-node

This directory vendors the `sq-node` code used by `pi-amp-pack` for subagent todo tracking.

Included here:

- compiled runtime files in `dist/`
- original TypeScript source files in `src/`
- `tsconfig.json`

Primary exports used by `pi-amp-pack`:

- `Queue`
- `resolveQueuePath`
- `mergeMetadata`
